GateFinder V1.0 for MSFS 2024 / MSFS 2020
=======================================

English
-------
Welcome to GateFinder! This tool automatically imports your current SimBrief flight plan and tells you exactly which gates/stands to use for departure and arrival based on the aircraft you are flying and its airline.

Installation:
1. "Community" folder: Copy the folder `wildbill75-gsx-gatefinder` into your MSFS `Community` folder. This gives you the in-game toolbar panel.
2. "GateFinder.exe": Run this application before or during your flight.

First Launch (Settings):
- Simbrief Username / Pilot ID: Enter your SimBrief username or Pilot ID.
- GSX Profile Path: If you use FSDreamTeam GSX, point this to your GSX Profiles folder (e.g. %APPDATA%\Virtuali\GSX\MSFS) to get ultra-precise gate sizing. If you don't use GSX, leave it blank; GateFinder will fall back to OpenStreetMap (OSM) to find your gates!
- Auto-Start: Launch GateFinder automatically when MSFS 2024 starts.
- Auto-Fetch: Automatically import your flight plan when the app starts.
- Scan Installed Airports: Click this to build a database of your local GSX profiles so they load instantly.
- Contribute Missing: Share your local GSX profile data with the community database!

Usage:
Once running, click "Import SimBrief Flight Plan". The app will display your Departure and Arrival gates. You can also view this directly inside the simulator by clicking the "GateFinder" icon in your MSFS Toolbar.

Français
--------
Bienvenue sur GateFinder ! Cet outil importe automatiquement votre plan de vol SimBrief et vous indique exactement quelles portes utiliser pour le départ et l'arrivée, en fonction de votre avion et de votre compagnie aérienne.

Installation :
1. Dossier "Community" : Copiez le dossier `wildbill75-gsx-gatefinder` dans votre dossier `Community` de MSFS. Cela ajoute le panneau dans la barre d'outils du jeu.
2. "GateFinder.exe" : Lancez cette application avant ou pendant votre vol.

Premier Lancement (Settings) :
- Simbrief Username / Pilot ID : Entrez votre nom d'utilisateur ou ID Pilote SimBrief.
- GSX Profile Path : Si vous utilisez FSDreamTeam GSX, indiquez le chemin vers vos profils GSX (ex: %APPDATA%\Virtuali\GSX\MSFS) pour des tailles de portes ultra-précises. Si vous n'avez pas GSX, GateFinder utilisera OpenStreetMap (OSM) pour trouver vos portes !
- Auto-Start : Lance GateFinder automatiquement au démarrage de MSFS 2024.
- Auto-Fetch : Importe automatiquement votre plan de vol au lancement de l'application.
- Scan Installed Airports : Cliquez ici pour construire une base de données de vos profils GSX locaux pour un chargement instantané.
- Contribute Missing : Partagez vos profils locaux avec la base de données de la communauté !

Utilisation :
Une fois lancé, cliquez sur "Import SimBrief Flight Plan". L'application affichera vos portes de départ et d'arrivée. Vous pouvez également consulter ces informations directement dans le simulateur en cliquant sur l'icône "GateFinder" dans la barre d'outils MSFS.
